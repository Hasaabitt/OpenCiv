# OpenCiv

An open source civilization inspired game.
I'm just trying to create a game similar to CIV 5 without networking issues to play with my friends!

Eventually, I'd like to add support modding through [LUAJ](https://www.gamedevelopment.blog/using-luaj-scripting-to-allow-modding-in-games/).
